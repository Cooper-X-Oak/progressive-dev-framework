# Progressive Dev Framework

渐进式开发框架 - 从快速原型到系统化代码的自然演进。

## 安装

```bash
npx progressive-dev-framework my-project
```

## 核心理念

**Demo → Refactor → Pattern**

1. 在 `demo/` 快速验证想法
2. 成熟后重构到 `src/`
3. 沉淀思路到 `patterns/`

## 项目结构

```
my-project/
├── demo/       # 快速原型和功能验证
├── src/        # 系统化的核心代码
├── patterns/   # 可复用的解决方案模式
└── docs/       # 架构决策记录
```

## 内置 Skills

```bash
/demo-design      # 设计 demo 的核心功能
/capture-pattern  # 沉淀可复用的模式
```

## 为什么使用

- **快速启动** - 不被过度设计拖累
- **自然演进** - 从原型到生产的平滑过渡
- **知识沉淀** - 像 OpenSpec 一样积累方法论
