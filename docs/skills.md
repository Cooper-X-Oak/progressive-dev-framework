# Skills 说明

## 已创建的 Skills

### /demo-design

帮助设计 demo 的核心功能，明确验证目标。

**使用场景**：

- 有一个想法，但不确定如何开始
- 需要明确验证目标
- 想要设计最小化实现

**工作流程**：

1. 理解需求和背景
2. 明确验证目标
3. 设计最小实现
4. 输出设计文档
5. 可选：创建 demo 文件骨架

### /capture-pattern

将管用的解决思路沉淀为可复用的模式。

**使用场景**：

- 发现了一个有效的解决方案
- 想要记录可复用的思路
- 需要建立知识库

**工作流程**：

1. 识别模式来源
2. 提取核心思路
3. 检查相关代码
4. 生成模式文档
5. 更新模式索引

## 使用方式

在对话中直接输入：

```
/demo-design
```

或

```
/capture-pattern
```
