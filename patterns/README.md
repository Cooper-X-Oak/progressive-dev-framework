# 思路模式库

沉淀可复用的解决方案，从 Demo 到 Pattern 的演进记录。

## 什么是"模式"？

- 从实际 Demo 中提炼的解决思路
- 展示从简单到复杂的演进路径
- 可复用的代码和方法论

### 开发方法论

## 如何使用模式

1. **遇到问题时**：浏览模式库，找到相似场景
2. **开始 Demo**：参考模式的 Demo 阶段快速实现
3. **需要重构**：参考模式的重构阶段提炼代码
4. **沉淀新模式**：使用 `/capture-pattern` 记录你的思路

## 贡献模式

使用 [模式模板](template.md) 创建新模式，包含：
- 问题描述
- Demo → 重构 → 模式的演进过程
- 实际代码示例
- 适用场景和关键决策
