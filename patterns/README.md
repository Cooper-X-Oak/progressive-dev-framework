# 思路模式库

## 目的
沉淀解决问题的管用思路，类似 OpenSpec 的理念。

## 什么是"模式"？
- 一个可复用的解决问题的思路
- 不是具体代码，而是方法论
- 经过实践验证有效的方案

## 模式格式
每个模式文件应包含：
```markdown
# 模式名称

## 问题
描述这个模式解决什么问题

## 解决思路
核心思路和步骤

## 适用场景
什么情况下使用这个模式

## 示例
具体应用示例

## 注意事项
使用时需要注意的点
```
