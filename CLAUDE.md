# Progressive Dev Framework

## 项目定位

这是一个**可复用的渐进式开发框架模板**，用于快速启动新项目。

## 核心理念

渐进式开发：从 demo 验证功能 → 提炼到系统化结构 → 沉淀可复用的思路模式

## 当前阶段

🚀 框架本身持续迭代中 - 欢迎贡献想法和改进

## 开发流程

1. **Demo 验证** - 在 `demo/` 中快速实现和测试想法
2. **功能提炼** - 将成熟的功能重构到 `src/`
3. **模式沉淀** - 把管用的思路记录到 `patterns/`
4. **持续演进** - 更新此文档记录项目约定

## 目录结构

- `demo/` - 功能验证和快速原型
- `src/` - 系统化的核心代码
- `patterns/` - 解决问题的思路模式库（类似 OpenSpec）
- `docs/` - 架构决策和演进记录

## 开发约定

- Demo 代码优先简单直接，不过度设计
- 功能验证通过后再考虑抽象和复用
- 每个有价值的解决思路都应该沉淀到 patterns/
- 保持 CLAUDE.md 简洁，详细内容放到对应目录

## 可用 Skills

- `/demo-design` - 设计 demo 的核心功能，明确验证目标
- `/capture-pattern` - 将管用的思路沉淀为可复用的模式

详见 [Skills 文档](docs/skills.md)
