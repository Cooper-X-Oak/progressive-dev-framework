# Contributing to Progressive Dev Framework

感谢你考虑为这个框架做贡献！

## 贡献方式

### 1. 报告问题
- 使用 GitHub Issues 报告 bug
- 描述清楚问题和复现步骤
- 说明你的使用场景

### 2. 提出改进建议
- 分享你的使用经验
- 提出新的 Skills 想法
- 建议新的模式或工作流程

### 3. 提交代码
- Fork 这个仓库
- 创建特性分支 (`git checkout -b feature/amazing-feature`)
- 提交你的改动 (`git commit -m 'Add some amazing feature'`)
- 推送到分支 (`git push origin feature/amazing-feature`)
- 创建 Pull Request

## 贡献指南

### 添加新的 Skill
1. 在 `.claude/skills/` 中创建新文件
2. 遵循现有 Skills 的格式
3. 在 `docs/skills.md` 中添加说明
4. 更新 `CLAUDE.md` 中的 Skills 列表

### 添加新的模式
1. 使用 `patterns/template.md` 创建新模式
2. 确保模式是通用的方法论，不是具体代码
3. 在 `patterns/README.md` 中添加索引
4. 按类别组织

### 改进文档
- 修正错别字和语法错误
- 补充使用示例
- 改进说明的清晰度

## 原则

- 保持简单和实用
- 注重可复用性
- 文档要清晰
- 遵循现有的结构和风格

## 问题？

如果有任何问题，欢迎创建 Issue 讨论。
